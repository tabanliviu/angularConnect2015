Protractor Styleguide

Recommendations

1. Put all e2e test files in one dedicated folder

2. Use one configuration file + multiple tasks

3. Never use XPATH

element(by.xpath("//*@id="calc"]/form/section/section[2]/div/div/div[1]/div/div/div[3]/div[2]/div[2]/div[1]/div[1]/h3"));

4. Page Objects

5. Make your tests independent at least at the file level

6. Prefer protractor locators

Locator Strategies

          
            <ul class="red">
            <li data-test-color-name>{{color.name | capitalize}}</li>
            <li>{{color.shade}}</li>
            <li>{{color.code}}</li>
            </ul>
          
      
          
            /* avoid */
            var nameElem = element.all(by.css('.red li')).get(0);

            /* recommended */
            var nameElem = element(by.binding('color.name'));

            /* the ncc way */
            var nameElem = element(by.css('[data-test-color-name]'));
